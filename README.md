# Guess The Number Game

A simple interactive **Guess the Number** game built using **HTML, CSS, and JavaScript**.

## Description
This game allows the user to guess a number between 1 and 20.  
- The game provides feedback if your guess is too high, too low, or correct.  
- Tracks the number of attempts.  
- Disables the guess button after 5 attempts or when the correct number is guessed.  
- Reset button starts a new game with a random number.

## Features
- Random number generation
- Input validation
- Dynamic feedback with colors
- Attempts counter
- Button disabling
- Reset functionality

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Clone the repository or download the ZIP file.
2. Open `index.html` in a web browser.
3. Enter a number in the input box and click **Guess**.
4. Click **Reset** to start a new game.
